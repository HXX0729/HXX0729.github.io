|Post Views:

静态变量就是用 static 修饰的变量

和Python一样，静态变量也成为类变量。是专属于类的，不会每个实例都有一个。因此他被存放在JVM的元空间

生存周期和类一样长，也就是整个程序的寿命

可以通过类名直接访问
